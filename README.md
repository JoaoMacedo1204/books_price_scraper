# Books Price Scraper

## Descrição:
Projeto de Web Scraping em Python que coleta títulos e preços de livros para tratamento de dados do site Books to Scrape.

## Funcionalidades:
- Extração de títulos e preços
- Tratamento de dados
- Exportação para CSV

## Tecnologias utilizadas:
- Python
- Requests
- BeautifulSoup
- Pandas

## Saída:
O script gera um arquivo 'livros.csv' com os dados coletados.
